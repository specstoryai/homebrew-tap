# SpecStory Homebrew Tap

Homebrew formulas for SpecStory tools.

## Formulas

| Formula     | Description                       |
|-------------|-----------------------------------|
| `stoa`      | Version control for coding agents |
| `specstory` | SpecStory CLI wrapper             |

## Installation

```zsh
brew tap specstoryai/tap

# Stoa CLI
brew install stoa

# SpecStory CLI
brew install specstory
```

## Upgrading

```zsh
brew update
brew upgrade stoa
```
